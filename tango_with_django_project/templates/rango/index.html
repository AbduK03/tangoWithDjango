Foundations Of Professional Software Engineering

Course convenor: Derek Somerville
Course start date: 1/8/2021

The aim of this course is to impart a breadth of knowledge about software engineering in a professional environment. This knowledge will cover ethical issues, design lifecycles, development methodologies and security factors. It will also develop their ability to critically assess software development design decisions. Students should also develop an understanding of common software development tools and packages on a conceptual level to make their transition into the workplace more efficient.

How to Learn a New Language

Course convenor: Matthew Barr
Course start date: 1/8/2021

The aim of this course is to provide students with the skills and knowledge to pick up a new programming language. The students will also be able to develop existing programming skills in order to embed sound programming language fundamentals and techniques for structuring programs appropriately. The course also teaches the basics of writing good clean code that is easy to read and maintain.
